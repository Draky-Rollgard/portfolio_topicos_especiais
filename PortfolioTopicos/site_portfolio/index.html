Tópicos Especiais

A disciplina de Tópicos Especiais abordou tecnologias modernas e cruciais para a formação em Engenharia de Computação, com foco no ecossistema Java.

Durante o período, os estudos foram direcionados ao desenvolvimento back-end corporativo moderno.

Spring Boot

Configuração e Facilidade

O Spring Boot revolucionou a forma como iniciamos projetos Java. A ferramenta "Spring Initializr" (start.spring.io) permite adicionar dependências iniciais facilmente.

  • Autoconfiguração: Menos XML, mais foco no código.
  • Tomcat Embutido: Execução autônoma de aplicações.
  • Injeção de Dependências: Inversão de controle com facilidade.
@SpringBootApplication
public class DemoApplication {
    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }
}

Padrão MVC e APIs

Desenvolvimento Web

Através do spring-boot-starter-webmvc, estruturamos a aplicação utilizando a arquitetura Model-View-Controller, ideal para APIs RESTful.

  • Controllers: Recebem as requisições HTTP (@RestController).
  • Rotas: Mapeamento flexível com @GetMapping, @PostMapping, etc.
  • JSON: Serialização automática de objetos.
@RestController
@RequestMapping("/api/usuarios")
public class UsuarioController {
    @GetMapping
    public List<Usuario> listar() {
        // Retorna lista
    }
}

Persistência de Dados

JPA e MySQL

A persistência foi tratada com spring-boot-starter-data-jpa e o conector do MySQL. Evitamos SQLs manuais com a abstração do Hibernate.

  • Entidades: Mapeamento de tabelas através de classes (@Entity).
  • Repositórios: Interfaces que herdam JpaRepository simplificam o CRUD.
  • Lombok: Redução de boilerplate (Getters/Setters automáticos).
@Entity
@Data // Lombok
public class Usuario {
    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;
    private String nome;
}
